Comandos que debes correr para iniciar el proyecto.
- Instalar todos los paquetes
```sh
npm install
```
- Ejecutar scripts para entrar en modo desarrollo
```sh
npm run build:dev
npm run build:media
npm run watch:dev
```
- Ejecutar scripts para entrar en modo producción
```sh
npm run build:prod
npm run watch:prod
```

```sh
localhost:3000
```
